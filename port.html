Proyectos

He desarrollado varios proyectos por propositos académicos, tanto en solitario como en grupo. Te invito a hecharles un ojo.

Como proyecto integrador de fin de año para el grado superior, se nos pidió ir creando un programa que disponga de un CRUD a lo largo del curso usando las diferentes tecnologías y herramientas que ibamos aprendiendo (conexión a base de datos, manejo de excepciones, interfaz de usuario...).

La temática que yo elegí fue la de un casino. En este, el usuario podría gestionar tanto los clientes que entren al casino a jugar, como los diferentes puestos de juegos (hay tanto blackjack como tragaperras para elegir).

Es el usuario quien deberá usar a estos clientes para poder apostar en los juegos. Tanto el cliente como el juego deberá tener dinero suficiente para hacer apuestas.

El programa cuenta con una interfaz intuitiva, una conexión a base de datos para no perder los datos al cerrar el programa y un sistema de inicio de sesión, donde cada usuario tendrá sus clientes, juegos y estadísticas.

Más que un programa de gestión, quise orientarlo más hacia lo que sería un videojuego, para que fuese entretenido tanto de programar como de usar. Quise haber integrado logros para cada usuario y más variedad de juegos a jugar, además de mas interacción con los juegos y clientes, quizás en un futuro lo actualize con nuevas mecánicas y un instalador más sencillo.

Similar al caso anterior, este fue un proyecto de fin de año del mismo curso, solo que para la asignatura de Desarrollo Web. Se nos pedía una aplicación web que se conectase a una api para recoger datos y almacenase información en el local storage.

En mi busqueda me encontré una sobre el clásico juego de "verdad o reto". Contaba con diferentes tipos de preguntas, con filtros, idiomas... Era justo lo que necesitaba para el proyecto.

Dentro de la aplicación, podras crear una sesión o jugar con una sesion guardada. Las sesiones son la configuración de la partida, donde marcas que tipos de preguntas quieres que salgan, el idioma, los filtros, los jugadores y demás.

Al entrar a jugar, uno de los jugadores al azar deberá elegir que pregunta responder, el programa le mostrará la pregunta y se pasará al siguiente jugar. Es una interfaz de juego sencilla pensada para que sea cómodo jugar.

PlayZone fue un proyecto grupal que se nos pidió en la asignatura de Desarrollo Web. Se requería el uso de temporizadores, drag and drop, listeneres y más funciones que vimos en el temario.

Entre tres compañeros decidimos hacer una pagina WEB de juegos, cada uno asignado a un juego diferente, mientras que supervisaba y ayudaba al resto del equipo en sus juegos.

Acabamos desarollando un quiz, uno juego de encontrar las parejas y un juego de estrategia similar al videojuego Plantas vs Zombies Heroes, evidentemente más simplificado.
